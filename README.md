## Installation

You can install the project by clone it via GitHub :

```bash
git clone https://github.com/ngodingbang/rwid-github-actions.git
npm install
```

## Testing

Run this command below to execute all test script using [Jest](https://jestjs.io).

```bash
npm run test
```
